# Yor-Smriti

1. What it is
An emotion-driven interactive web experience focused on memories, timelines, and narrative storytelling.

2. Why you built it
Memory-based messages can feel flat when they are delivered as plain text. Yor-Smriti explores how interaction, pacing, and visual tone can make a private message feel more intentional.

3. How it works
The GitHub project is a TypeScript web experience with themes, templates, audio support, shareable links, and timeline-oriented storytelling surfaces.

4. What you learned
Emotional interfaces need restraint. The strongest interaction is the one that supports the message without overwhelming it.
