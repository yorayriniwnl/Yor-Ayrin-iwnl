# Mentor-Mentee System

1. What it is
A Python platform for managing mentor-mentee relationships with matching logic, a Flask API layer, SQLite-backed data, and a Tkinter desktop interface.

2. Why you built it
Mentorship programs can become spreadsheet-heavy and hard to coordinate. The project explores a cleaner way to organize people, compatibility, and operational workflows.

3. How it works
Core matching logic is shared across the app. Flask exposes API workflows, SQLite supports local persistence, and Tkinter provides a desktop GUI for local institutional use.

4. What you learned
Simple operational tools still need thoughtful data modeling. The useful part is not only the interface, but the reusable logic underneath it.
