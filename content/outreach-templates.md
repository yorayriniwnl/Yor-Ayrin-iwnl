# Outreach Templates — Short, copy-ready messages

Use these templates to apply, follow up, and message contacts. Keep each message short and include a portfolio link.

1) ATS / Cover opening (1–2 lines)

Subject: Application — [Role] — [Your Name]

Hi [Hiring Manager],

I built [Yor Zenith], which [one-line impact]. I'm excited about [Company]'s work on [area]; my background in systems thinking and delivery fits this role. Resume and portfolio: [site link].

2) Email to recruiter / hiring manager (short)

Subject: Quick note — [Role] at [Company]

Hi [Name],

I applied for [Role]. Briefly: I led the architecture and delivery for Yor Zenith, which [metric/impact]. If you're open, I'd welcome a 20-minute conversation to share details and fit. Portfolio: [site link].

3) LinkedIn connection request (50–100 chars)

Hi [Name] — I enjoyed your work on [topic]. I build product systems (Yor Zenith). Mind if I connect?

4) LinkedIn follow-up after connect

Hi [Name], thanks for connecting. I recently led Yor Zenith (brief: [one-line]). If you're hiring or can introduce me to the relevant team, I'd appreciate a quick chat. Portfolio: [site link].

5) After application follow-up (3–7 days)

Hi [Name],

Following up on my application for [Role]. I led Yor Zenith (one-line result). If helpful, I can send a short summary of the plan I would start with in this role. Thanks for considering.

6) Interview thank-you / post-screen

Hi [Name],

Thanks for the quick chat today — I appreciated learning about [topic]. As discussed, here's a short note on how I'd approach [one concrete item]. Portfolio: [site link]. Looking forward to next steps.

Notes
- Tailor the first sentence to a specific company detail.
- Keep each message ≤ 5 short sentences. Always include a portfolio link and one-line impact.
