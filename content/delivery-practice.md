# Delivery Practice — Quick Guide

Rules
- Speak slowly and deliberately.
- No filler words (uh, um, like, you know).
- Pause 1–2 seconds between each point.

How to use the recorder page
1. Open `/delivery/index.html` in your dev server or via the `public` folder (e.g., http://localhost:3000/delivery/index.html).
2. Paste a short script into the textarea (1–4 lines; one point per line).
3. Press `Start Recording`. A 3s count-in will begin.
4. Speak each line slowly, pause 1–2 seconds between points.
5. Press `Stop`, then `Play` to review. Download the take if you want.

Optional live feedback
- Click `Live Transcribe` for an experimental real-time transcript (Chrome/Edge only). The page highlights common filler words when detected.

Practice routine
- Do 3 takes of the same script. Pick the best take and compare what changed.
- Record a short (30–60s) answer to the question: "Why hire me for this role?" — focus on clarity and pauses.
