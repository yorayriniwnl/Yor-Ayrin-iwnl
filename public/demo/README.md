Demo Player — Quick Recording Guide

Open `/demo/index.html` (the file is served from your site at `https://<your-site>/demo/index.html` or `http://localhost:3000/demo/index.html` when running locally).

How to record:

- Recommended: Use OBS or your system screen recorder.
- Set output resolution to 1280x720 (or 1920x1080) and 30 fps.
- Open the demo page in a browser, make it full-screen (F11) and start recording.
- The demo auto-plays a 30s sequence (Hero → Projects → Yor Zenith).

Notes:
- This is an auto-playing presentation designed for fast screen recording. If you need an MP4 file generated programmatically, install `ffmpeg` locally and I can provide exact ffmpeg commands to assemble a video from the screenshots.
